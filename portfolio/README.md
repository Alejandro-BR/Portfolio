# React + TypeScript + Vite

### ⚙️ Comandos básicos:

**Instalar dependencias:**
```bash
npm install
```

**Levantar el servidor de desarrollo:**
```bash
npm run dev
```

**Crear el build de producción:**
```bash
npm run build
```

**Previsualizar el build de producción localmente:**
```bash
npm run preview
```
